📖 How Network Attendance Works
🛰️ Setup Process:
  1. Create an ESP32 device entry
  2. Create a network session
  3. ESP32 creates WiFi hotspot
  4. Students connect to WiFi
  5. Attendance automatically recorded
📱 Student Process:
  1. Join ESP32 WiFi network
  2. Stay connected during class
  3. Attendance automatically verified
  4. No manual check-in needed
