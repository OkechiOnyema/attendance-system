Instructions for Students
📱 How to Mark Attendance:
  1. Connect to ESP32 WiFi network (no password)
  2. Open browser and go to: 192.168.4.1
  3. Enter your matric number and name
  4. Click "Mark Attendance"
  5. You'll see confirmation if successful
🔍 Troubleshooting:
  • Make sure you're enrolled in this course
  • Check that your matric number is correct
  • Ensure you're connected to the right WiFi
  • Try refreshing the page if it doesn't work
